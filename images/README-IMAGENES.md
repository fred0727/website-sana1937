## Especificaciones Técnicas

### Formatos Recomendados
- **Logos:** PNG con transparencia
- **Fotos:** JPG optimizadas (calidad 85%)
- **WebP:** Para mejor rendimiento (opcional)

### Tamaños y Optimización
- **Peso máximo por imagen:** 300KB
- **Resolución:** 72 DPI para web
- **Compresión:** Optimizar para web sin perder calidad

### Colores y Estilo
- **Paleta principal:** Verdes (#2d5016, #4a7c59, #6b9b47)
- **Acento:** Dorado (#d4af37)
- **Estilo:** Moderno, limpio, natural, profesional
- **Evitar:** Elementos muy saturados o artificiales

## Instrucciones para Implementar

1. **Descargar los logos** desde los enlaces de Canva proporcionados
2. **Renombrar los archivos** según la nomenclatura especificada
3. **Optimizar las imágenes** para web (comprimir sin perder calidad)
4. **Colocar en la carpeta** `/images/` del proyecto
5. **Verificar que se muestren correctamente** en todas las páginas
