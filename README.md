# SANA 1937 - Landing Page

## Descripción del Proyecto

Landing page profesional para SANA 1937, marca peruana dedicada al bienestar natural con más de 85 años de tradición familiar.

## Estructura del Proyecto

```
sana1937/
├── index.html              # Página principal
├── css/
│   └── styles.css          # Estilos principales
├── js/
│   └── main.js            # JavaScript principal
├── images/                # Imágenes y recursos visuales
├── pages/                 # Páginas adicionales
│   ├── nosotros.html      # Historia y valores de la empresa
│   ├── productos.html     # Catálogo de productos (Coming Soon)
│   ├── marcas.html        # Marcas aliadas (Coming Soon)
│   ├── smoothies.html     # Menú de smoothies (Coming Soon)
│   ├── franquicia.html    # Oportunidades de franquicia
│   └── contacto.html      # Formulario de contacto
└── README.md              # Este archivo
```

## Características Implementadas

### ✅ Header Sticky
- Navegación fija inspirada en Gymshark
- Logo central de SANA 1937
- Menú responsive con hamburger para móvil
- Orden: Productos | Marcas | Nosotros | **LOGO** | Smoothies | Franquicia

### ✅ Footer Responsivo (3 Columnas)
- **Columna 1:** Redes sociales (Facebook e Instagram)
- **Columna 2:** Información corporativa "SANA 1937 - Quiénes Somos"
- **Columna 3:** Servicio al Cliente - Contacto

### ✅ Página Principal
- Banner full-width con logo centrado
- Sección "Nuevos Productos"
- Sección "Quiénes Somos - Smoothies"
- Valores de la marca
- Call-to-action

### ✅ Página Nosotros
- Historia completa desde 1937
- 3 secciones con fotos y textos
- Valores y certificaciones
- Línea de tiempo de la empresa

### ✅ Página Franquicia
- Título: "Conviértete en un dueño"
- 4 subsecciones principales:
  - Marketing y publicidad
  - Proveedores
  - Eficiencia
  - Proceso (6 pasos detallados)
- Requisitos e inversión
- Testimonios de franquiciados

### ✅ Página Contacto
- Formulario con campos obligatorios:
  - Nombre Completo
  - Email
  - Asunto
  - Mensaje
- Información de contacto
- Horarios de atención
- Enlaces a WhatsApp
- Preguntas frecuentes

### ✅ Páginas Coming Soon
- Productos naturales

- **Negro:** `#000000` (Para acentos y hover states)
- **Grises:** `#333333`, `#666666`, `#999999` (Para textos y elementos)

## Instalación y Uso

1. Clonar o descargar el proyecto
2. Abrir `index.html` en un navegador web
3. Para desarrollo local, usar un servidor HTTP simple

### Servidor Local (Opcional)
```bash
# Con Python 3
python -m http.server 8000

# Con Node.js (http-server)
npx http-server

# Con PHP
php -S localhost:8000
```
